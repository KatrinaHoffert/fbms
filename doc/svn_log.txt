r163 | m.l.hoffert@gmail.com | 2013-11-02 12:51:04 -0600 (Sat, 02 Nov 2013) | 8 lines

Implemented a new utility function: FileOp.convertPath(Path), which converts a backup directory path
to a live directory path and vice versa.

For example, if file corresponds to the path located at %liveDir%/a/b/c, then
FileOp.convertPath(file) will output %backupDir%/a/b/c.

This is useful when copying files between the live and backup directory, as we can simply do
FileOp.copy(file, FileOp.convertPath(file)).

** IMPORTANT: FileOp.convertPath() will return null if the input path is not in the backup or live
directory. If there's a chance that the path passed into this function is invalid, be sure to check
that the output of FileOp.convertPath() is not null.
